survival.io.writearff¶

survival.io.writearff(data, filename, relation_name=None, index=True)¶

Write ARFF file

Parameters:

data_frame : pandas.DataFrame

DataFrame containing data

filename : string or file-like object

Path to ARFF file or file-like object. In the latter case, the handle is closed by calling this function.

relation_name : string, optional, default: “pandas”

Name of relation in ARFF file.

index : boolean, default True

Write row names (index)
